# ddgr-skill

DuckDuckGo web search skill for AI coding agents using [ddgr](https://github.com/jarun/ddgr).

## Installation

```bash
npx skills add ysm-dev/ddgr-skill
```

## Prerequisites

Install ddgr:

```bash
# macOS
brew install ddgr

# pip
pip install ddgr

# Other: https://github.com/jarun/ddgr#installation
```

## Usage

Once installed, your AI agent will use this skill when you ask to:

- Search the web
- Look something up online
- Find documentation or tutorials
- Get current information
- Use DuckDuckGo Bangs (`!w` for Wikipedia, `!so` for StackOverflow, etc.)

## Examples

```
Search for React hooks best practices
```

```
Look up the latest Node.js release
```

```
Find Python asyncio documentation
```
